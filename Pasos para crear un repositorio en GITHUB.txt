1- Crear el repositorio en GITHUB

2- Abrir GIT

3- Escribimos un ""git status"

4- Despues un "git add ."

5- Volvemos a escribir un "git status" para comprobar que todo se alla subido

6- Despues hacemos un commit con "git commit -m "Nombre del commit" "

7- Luego haces un branch con "git branch -M main"

8- Seguido copiamos el link que nos habra echo GITHUB al crear el repositorio

9- Y acabaremos con un "git push -u origin main"
